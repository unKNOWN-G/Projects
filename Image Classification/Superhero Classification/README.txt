Super Hero Classification

In this project we are trying to classify the SuperHero in image toone of the 4 super heros 'Captain America' , 'Thor' , 'Hulk' , 'Iron Man'
This is a Multi Class classification
Convolution Neural Net has been used
Data Augumentation is done using Image data generator libary
The training set and the test set can be found at the : https://drive.google.com/drive/folders/1nvUEeXKGW_mQHZK9tGNrnzNKZ3BHsHIF?usp=sharing
